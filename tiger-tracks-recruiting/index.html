⚡ PhantomBuster / Zapier Auto-Import

Connect PhantomBuster or Zapier to automatically push LinkedIn prospects into the recruiting tool. Prospects are de-duplicated and auto-scored against all open roles.

🤖 PhantomBuster Setup

  1. Create a LinkedIn Search Export or Sales Navigator Search phantom
  2. Configure your search filters (title, location, industry, etc.)
  3. In Phantom settings, go to "Notify" or "Webhooks"
  4. Set webhook URL to the endpoint below
  5. Run the phantom - results auto-flow here

⚡ Zapier Setup

  1. Create a Zap with your trigger (Google Sheet, PhantomBuster, etc.)
  2. Add action: "Webhooks by Zapier" > POST
  3. Set URL to the endpoint below
  4. Payload Type: JSON
  5. Map fields: name, title, company, linkedinUrl, location, email
